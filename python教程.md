# Python教程

对于初学者来讲，Python的学习可以分为以下三个阶段：

* 掌握基本语法

此阶段最为快速的方法就是找一个高质量的教程或者教材，有计划的通读。

基本语法包括数据类型、流程控制、函数编写等基础内容，也包括pandas、numpy等常用库（具体选择哪些库可以根据自己的专业领域来确定）的常用操作。

* 长期大规模的练习

对于一个有编程基础的人来说（至少有C语言基础，掌握常见的数据结构和算法），掌握了Python的基本语法之后，基本上就算是入门了。接下来想要进一步提高，唯一的方法就是 ** 进行大规模的实战训练 ** 。

而进行大规模实战训练最好的方式就是将Python作为自己的工作语言。所谓工作语言，就是之前你用Excel、R、SAS、C、Java等来完成的工作，现在都要用Python来完成，虽然这样做很多时候会降低你的工作效率，但是这是值得的。

在这一阶段，如果遇到具体的问题（比如如何对一个数组排序），可以通过Google或百度来解决，先把任务完成。等任务完成之后，可以对自己遇到的问题涉及的模块进行深入的有针对性的学习。


* 深入有针对性的学习

此阶段主要是为了系统性的解决练习阶段出现的问题，比如：

Example1：练习阶段涉及到了求解一个二次规划问题，当时只是通过百度找到了响应的函数解决，那么此时就可以直接去找scipy的官方文档，深入的学习python求解优化问题都有哪些方法，有哪些函数，每个参数代表什么含义等；

Example2：练习阶段经常遇到时间数据格式处理的问题，那么此时就可以把datetime库拿出来认真学习一下，并总结时间格式数据的常见问题以及解决方法；

Example3：练习阶段涉及到Logistic回归，当时只是通过百度依照案例代码解决了问题。那么此时就可以把sklearn库拿出来好好学习一下。

## 推荐书目

在第一阶段 **掌握基本语法**的过程中，可以参考以下书目（可根据自己的专业领域有所选择）：

* Python基础教程(Beginning Python)
* 廖雪峰的Python教程（https://www.liaoxuefeng.com/wiki/001374738125095c955c1e6d8bb493182103fac9270762a000/）
* 利用Python进行数据分析(Python for Data Analysis)
* Python金融大数据分析(Python for Finance)

注意：很多书目的内容比较全面，此时一定要根据自己的工作方向或者兴趣点进行取舍。比如如果之后想利用Python进行数据分析或者机器学习，那么就可以适当跳过一些开发相关的内容。



在第三阶段**深入有针对性的学习**的过程中，可以学习一下库：

* numpy
* pandas
* matplotlib
* scipy
* sklearn
